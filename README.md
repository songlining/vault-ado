# Azure DevOps + HashiCorp Vault Workload Identity Federation

This repository contains a Terraform-based reference implementation for integrating Azure DevOps pipelines with HashiCorp Vault using Azure DevOps Workload Identity Federation and the Vault Azure auth backend.

The detailed implementation report is here:

- [Terraform implementation guide](terraform/README.md)

It covers the architecture, runtime authentication flow, Azure DevOps service connection, Microsoft Entra federated credential, Vault Azure auth configuration, pipeline template, and security considerations.
